My research centres on **acoustic and elastic metamaterials** — engineered periodic structures whose wave behaviour is set by geometry rather than bulk material. I model wave propagation and band-gap formation in 2D metamaterial unit cells using finite element methods, primarily with the open-source [Firedrake](https://www.firedrakeproject.org/) framework and the [PETSc](https://petsc.org/) solver stack.
